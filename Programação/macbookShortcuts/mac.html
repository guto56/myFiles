Comandos Macbook

Guia de comandos rápidos usando teclas do Macbook

Comandos Gerais

  • Copiar: Command (⌘) + C
  • Colar: Command (⌘) + V
  • Recortar: Command (⌘) + X
  • Desfazer: Command (⌘) + Z
  • Refazer: Command (⌘) + Shift + Z
  • Selecionar tudo: Command (⌘) + A
  • Salvar: Command (⌘) + S
  • Imprimir: Command (⌘) + P
  • Fechar janela: Command (⌘) + W
  • Fechar aplicativo: Command (⌘) + Q

Comandos de Navegação

  • Ir para a área de trabalho: Command (⌘) + F3
  • Alternar entre janelas do mesmo app: Command (⌘) + ~
  • Alternar entre apps abertos: Command (⌘) + Tab
  • Mostrar ou ocultar o Launchpad: F4 ou Command (⌘) + Space (para Spotlight)
  • Abrir Finder: Command (⌘) + N (dentro do Finder)

Captura de Tela

  • Capturar toda a tela: Command (⌘) + Shift + 3
  • Capturar uma área selecionada: Command (⌘) + Shift + 4
  • Capturar uma janela específica: Command (⌘) + Shift + 4 e depois pressione a barra de espaço
  • Capturar a Touch Bar (se houver): Command (⌘) + Shift + 6

Comandos do Sistema

  • Mostrar/ocultar o Dock: Command (⌘) + Option + D
  • Abrir as Preferências do Sistema: Command (⌘) + Espaço e digitar "Preferências do Sistema"
  • Forçar encerramento de aplicativos: Command (⌘) + Option + Esc
  • Bloquear a tela: Command (⌘) + Control + Q
  • Desligar/reiniciar/suspender o Mac: Control + Eject (ou Power Button)

Comandos com a Seta para Cima (↑)

  • Ir para o início do documento ou página: Command (⌘) + seta para cima
  • Mostrar a linha de comando anterior no Terminal: Seta para cima
  • Scroll para cima em uma página ou documento: Seta para cima
  • Navegar entre pastas no Finder: Command (⌘) + seta para cima

Comandos com o Símbolo `^` (Control)

  • Mover o cursor para o início da linha no Terminal: Control + A
  • Mover o cursor para o final da linha no Terminal: Control + E
  • Apagar uma palavra no Terminal: Control + W
  • Limpar a tela do Terminal: Control + L
