Sobre o MAR:

O Menu de Acesso Rápido é o atalho perfeito para quem busca praticidade e eficiência no dia a dia. Com ele, você tem acesso imediato a uma série de funções essenciais do seu iPhone, tudo em um único lugar!

Ainda por cima, ele é aberto, ou seja, você pode editar, adicionar, remover, diversas funções e adaptar para o seu agrado pessoal.

Como adicionar?

Adicionar o MAR é tão simples quanto utilizá-lo, basta clicar no botão abaixo ou se estiver no Computador clicar na letra 'L' para ser redirecionado.

O Link para adicionar é totalmente seguro, é criado diretamente da Apple, do sistema IOS, então você não será redirecionado para sites diversos para poder conseguir adicionar.
